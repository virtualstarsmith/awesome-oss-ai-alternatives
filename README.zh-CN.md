## 2. ChatGPT / 聊天 UI

- [**open-webui/open-webui**](https://github.com/open-webui/open-webui) — 自托管 ChatGPT 仿作天花板，138K★。 *(alt: ChatGPT)*

- [**khoj-ai/khoj**](https://github.com/khoj-ai/khoj) — 个人 AI 第二大脑，对接 Obsidian / Notion / 邮件。 *(alt: NotebookLM / Notion AI)*

## 8. 本地推理引擎

上面所有「本地」项目的底座。不想调用 OpenAI 时，就靠它们。

- [**ollama/ollama**](https://github.com/ollama/ollama) — 本地跑 LLM 事实标准。 *(alt: OpenAI API for local)*
- [**ggml-org/llama.cpp**](https://github.com/ggml-org/llama.cpp) — 大半本地 LLM 工具的 C++ 内核。 *(infra)*
- [**vllm-project/vllm**](https://github.com/vllm-project/vllm) — 高吞吐 LLM 推理服务。 *(alt: OpenAI / Together.ai serving)*
- [**mudler/LocalAI**](https://github.com/mudler/LocalAI) — OpenAI 兼容协议的本地替身。 *(alt: OpenAI API)*
- [**mozilla/llamafile**](https://github.com/mozilla/llamafile) — Mozilla 出品，将 LLM 打包为单个可执行文件分发运行。 *(alt: OpenAI API for local)*
- [**huggingface/text-generation-inference**](https://github.com/huggingface/text-generation-inference) — HuggingFace 出品的生产级 LLM 推理服务，支持连续批处理。 *(alt: OpenAI API serving)*

## 11. 浏览器自动化 + OCR 文档解析
