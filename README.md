## 8. Local Inference Engines

The runtimes that power everything above when you don't want to call OpenAI.

- [**ollama/ollama**](https://github.com/ollama/ollama) — De-facto standard for running LLMs locally, 172K★. *(alt: OpenAI API for local)*
- [**ggml-org/llama.cpp**](https://github.com/ggml-org/llama.cpp) — The C++ kernel behind most local-LLM tools. *(infra)*
- [**vllm-project/vllm**](https://github.com/vllm-project/vllm) — High-throughput LLM serving with PagedAttention. *(alt: OpenAI / Together.ai serving)*
- [**mudler/LocalAI**](https://github.com/mudler/LocalAI) — Drop-in OpenAI-compatible API for local models. *(alt: OpenAI API)*
- [**mozilla/llamafile**](https://github.com/mozilla/llamafile) — Distribute and run LLMs as a single file, by Mozilla. *(alt: OpenAI API for local)*
- [**huggingface/text-generation-inference**](https://github.com/huggingface/text-generation-inference) — Production LLM serving with continuous batching, by HuggingFace. *(alt: OpenAI API serving)*
